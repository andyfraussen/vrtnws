# vrtnws

VRT NWS terminal reader. Browse the latest news and sports from [vrtnws.be](https://www.vrt.be/vrtnws/nl/) and [sporza.be](https://sporza.be) without leaving your terminal.

## Install

```bash
brew tap andyfraussen/tap
brew install vrtnws
```

## Usage

```bash
vrtnws
```

### Keys

| Key | Action |
|-----|--------|
| `↑` / `↓` | Navigate articles |
| `Enter` | Open article |
| `ESC` / `b` | Back to list |
| `O` | Open full article in browser |
| `Tab` | Switch between Nieuws and Sport |
| `h` | Help |
| `q` | Quit |

## Sections

Nieuws (VRT NWS), Sport (Sporza).

## Requirements

macOS with Homebrew. Python 3.12 is installed automatically by Homebrew.
